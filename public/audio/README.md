# Ambient audio

`ambience.mp3` is a soft looping placeholder (wind through tall grass with a
faint high shimmer). It is muted by default and only plays when a visitor taps
the Ambience toggle in the lower-left corner.

## Replace it

Drop in your own loop and keep the same name and path:

    public/audio/ambience.mp3

A seamless 20 to 40 second loop works best. Suggested content: wind through
grass, distant birds, subtle wilderness ambience. No code changes are needed.
